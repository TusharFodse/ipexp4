Finding Roots of a Quadratic Equation

Problem Statement

Calculate the real roots of a quadratic equation using JavaScript.

Code Implementation


function findRealRoots(a, b, c) {
    let discriminant = b * b - 4 * a * c;
    if (discriminant > 0) {
        let root1 = (-b + Math.sqrt(discriminant)) / (2 * a);
        let root2 = (-b - Math.sqrt(discriminant)) / (2 * a);
        return `Roots are ${root1} and ${root2}`;
    } else if (discriminant === 0) {
        let root = -b / (2 * a);
        return `Root is ${root}`;
    } else {
        return "No real roots";
    }
}

console.log(findRealRoots(1, -3, 2)); // Outputs: Roots are 2 and 1
console.log(findRealRoots(1, 2, 1));  // Outputs: Root is -1
console.log(findRealRoots(1, 1, 1));  // Outputs: No real roots

Result

The function accurately calculates the real roots of the quadratic equation based on the discriminant.
