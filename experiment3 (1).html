Studying Functions in JavaScript

Problem Statement

Explore functions as objects, arrow functions, inner functions, and anonymous functions in JavaScript.

Code Implementation


// Function as Object
function greet() {
    console.log("Hello, World!");
}
greet.language = "English";
console.log(greet.language); // Outputs: English

// Arrow Function
const sum = (a, b) => a + b;
console.log(sum(5, 3)); // Outputs: 8

// Inner Function
function outer() {
    function inner() {
        console.log("Inner function");
    }
    inner();
}
outer(); // Outputs: Inner function

// Anonymous Function
setTimeout(function() {
    console.log("This is an anonymous function");
}, 1000);

Result

This experiment demonstrates various types of functions in JavaScript, showcasing their versatility and applications.
